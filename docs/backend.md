# Backend & Datenverarbeitung

Dieses Kapitel beschreibt die Server- und Datenlogik des Systems.

## Inhalt
- Entwicklung eines REST- oder MQTT-basierten Backends
- Verarbeitung von Sensordaten und Events
- Speicherung in Datenbanken oder Object Storage (z. B. MinIO)
- Deployment via Docker / k3s Kubernetes Cluster

## Ziel
Zentrale Verwaltung und Verarbeitung aller Daten aus dem Edge-System.
