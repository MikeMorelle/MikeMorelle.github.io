# Monitoring & System Health

Dieses Kapitel beschreibt die Überwachung der gesamten Infrastruktur.

## Inhalt
- Einrichtung von Prometheus oder CheckMK
- Visualisierung mit Grafana
- Monitoring von CPU, RAM, Netzwerk und Temperatur
- Erkennung von Systemfehlern und Ausfällen

## Ziel
Echtzeit-Überwachung aller Nodes und Services im Cluster.
