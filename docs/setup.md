# Setup & Infrastruktur

Dieses Kapitel beschreibt den Aufbau der Edge-Computing-Infrastruktur.

## Inhalt
- Installation von Raspberry Pi OS / Ubuntu
- Konfiguration der Raspberry Pi 5 und Raspberry Pi 3 Nodes
- Einrichtung der Raspberry Pi AI Camera und AI HAT+
- Netzwerk-Setup (LAN, SSH, statische IPs)

## Ziel
Eine funktionierende Edge-Computing-Umgebung mit mehreren Sensor-Nodes und zentraler Verwaltung bereitzustellen.
