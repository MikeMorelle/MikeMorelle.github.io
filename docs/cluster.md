# Cluster & High Performance Computing

Dieses Kapitel behandelt den Aufbau und die Nutzung des Raspberry Pi Clusters.

## Inhalt
- Aufbau des Raspberry Pi Clusters (Master/Worker Architektur)
- Installation von MPI (Message Passing Interface)
- Durchführung von Parallelberechnungen
- Skalierungstests mit mehreren Nodes

## Ziel
Demonstration der Rechenleistung und Skalierbarkeit eines Edge-Clusters.
