# Frontend & Dashboard

Dieses Kapitel beschreibt die Benutzeroberfläche des Systems.

## Inhalt
- Entwicklung eines Dashboards mit React oder Vue.js
- Darstellung von Events, Logs und Kamerabildern
- Visualisierung von Standort- oder Cluster-Status
- Kommunikation mit Backend über REST API oder MQTT

## Ziel
Benutzerfreundliche Echtzeit-Übersicht über das gesamte System.
